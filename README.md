# LottoTeamProject
Spring boot와 Vue.js를 활용한 웹 쇼핑몰 프로젝트입니다.

<br>

## 📌주요 기능
#### 1. 카테고리별 상품 검색 및 정렬 기능
 - 검색창을 활용한 상품 검색
 - 카테고리별 상품 정렬
#### 2. 장바구니 기능
 - 원하는 상품 장바구니 담기
#### 3. 주소 검색 기능
 - 회원가입 시 회원 주소 입력
#### 4. 결제 API
 - TOSS API를 연동하여 상품 결제
#### 5. 소셜 로그인
 - Kakao 소셜 로그인으로 회원가입 및 로그인 
#### 7. 상품/주문내역 등 쇼핑몰 관리 기능
 - 관리자 페이지를 통한 상품 등록, 조회  수정
 - 주문 환불 관리

<br>
